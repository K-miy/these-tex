\begin{block}{Imaginons une ferme automatisée ...}
\begin{itemize}
\visible<2->{\item Où travaille un robot ... }
\visible<3->{\item devant réaliser plusieurs tâches journalières:}
\visible<4->{\begin{itemize}
    \item Nourrir les poules
    \item Traire les vaches
    \item Arroser le potager
    \item (etc ...)
    \end{itemize}}

\visible<6->{\begin{itemize}
    \ifleche Les tâches qu'il réalise peuvent parfois prendre plus de temps
    \ifleche Il a en plus des chances d'échouer (ou de mal réaliser ces tâches)
    \end{itemize}}
\visible<7->{\item Comment modéliser ce problème de \emph{planification} ?}
\end{itemize}
\end{block}

\begin{frame}

 \frametitle{Résultats théoriques}

\visible<+->{
\begin{exampleblock}{Gain dans un problème d'allocation séquentiel}
Dans un problème d'allocation de ressources à des tâches, le gain en facteur de branchement de l'utilisation de la contrainte \emph{allDiff} qui imposent que toutes les ressources soient appliquées à des tâches différentes est égal à:
\[ Gain = \frac{|T|^{|X|}}{C_{|T|}^{|X|}}\]
\end{exampleblock}}

\visible<+->{
\begin{alertblock}{Complexité du modèle en pire cas}
Trouver une politique pour un \textsc{m}a\textsc{csp}, qui obtiendrait une récompense espérée d'au moins $C$, est \#\textsc{p}.
\end{alertblock}
}

\visible<+->{
\begin{block}{}
Mais cela fonctionne très bien \emph{en pratique} puisque le nombre de variables est très inférieur au nombre d'états.
\end{block}
}
\end{frame}

\begin{small}
\begin{eqnarray}
  \bo_i^\star &\gets& \la o_i, \bo_{-i}^\star \ra \notag\\
  \mbox{où }\bo_{-i}^\star &=& \arg \max_{\bo_{-i} \in \JObs{-i}} \Obs(\la o_i,\bo_{-i}\ra|\ba,s')\bel^t_i(s') \notag\\
  \bel_i^{t+1}(s') &=& \frac{\Obs(\bo_i^\star|\ba,s') \sum_{s\in\Sta} \Tra(s'|s,\ba) \bel_i^t(s)}{\sum_{s,s'\in\Sta}{\Obs(\bo_i^\star|\ba,s') \Tra(s'|s,\ba) \bel_i^t(s)}}
\end{eqnarray}
\end{small}
}

\begin{large}
\begin{equation}
\vspace{-13pt}
\bel_i^{t+1}(s') = \max_{\bo_{-i}} \bel_i^{t+1}(s')
\end{equation}
\end{large}}
\end{block}}

\end{equation}
\end{small}
}
\only<4>{
\begin{large}
\begin{equation}

\end{equation}
\end{large}}
\end{block}}

\begin{block}{}
    \begin{itemize}

    \end{itemize}
